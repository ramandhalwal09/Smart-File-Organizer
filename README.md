## Smart File Organizer
Smart File Organizer using Python

A Python automation tool that automatically organizes files into folders based on file extensions.

## Features
- Automatic file sorting
- Organizes Images, Videos, PDFs, Music, etc.
- Handles duplicate filenames

## Technologies Used
- Python
- os
- shutil
- datetime

## How to Run
1. Set folder path
2. Run organizer.py
